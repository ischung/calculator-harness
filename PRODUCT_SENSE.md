## 제품 비전

> **"수학은 거짓말하지 않는다. 우리 플랫폼도 그래야 한다."**

사칙연산 플랫폼은 단 하나의 약속을 지킨다:
**정확한 계산, 안전한 에러 처리, 명확한 피드백.**

## 제품 결정 기록 (Decision Log)

### PDR-01: 에러를 throw하지 않고 Result 타입을 반환한다

- **결정**: `divide(10, 0)` → `{ ok: false, error: 'DIVISION_BY_ZERO' }` (throw 아님)
- **이유**: throw는 제어 흐름을 예측 불가능하게 만든다. Result 타입은 에러를 타입 시스템이 강제한다.
- **트레이드오프**: 코드가 약간 verbose해지나, 안전성이 압도적으로 높다.

### PDR-02: 계산 결과는 부동소수점 그대로 반환한다

- **결정**: 반올림 없이 JS 숫자 그대로 반환
- **이유**: 반올림 정책은 UI 레이어의 책임이다. core/는 수학적 사실만 반환한다.
- **예외**: NaN, Infinity는 에러로 처리한다.
